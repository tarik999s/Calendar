1. To start program use Calendar.jar file

2. To compile call:

   javac com/taras/my_puzzle/main/MainClass.java

3. To run, first copy one or two images in JPEG or GIF format into

   com/taras/my_puzzle/images

   Then create the file "puzzleImages.txt" in the same directory,
   and put the names of the image files in this txt file, in the format:

   [image] [Name of the image]

   Example:

	flag.jpg The image of the flag
	moon.gif The image of the moon

4. To run now call:

   java com.taras.my_puzzle.main.MainClass
